Web Danielspage 1. Word definitions

Decimal

The decimal system is used to represent numbers, using digits from 0-9. For example, 21. Each column is 10 times larger than the last.

Binary

Binary is a system for representing numbers using 1 & 0. for example, 10101. Each column is twice as big as the last, meaning that the 10101 is the same as 21.

Bit

Bits are the smallest amount of information that there is in a computor. It is a switch and is either off or on.

Byte

8 bits.

protocol

A protocol is a set of rules or standards that needs to be used to communicate between computors. If computors do not have the same protocol when communicating, they can not communicate.

Analog

An analog value can have any value between 0 and 1. It could be used to fade a light on or off instead of an on or off switch, or make a motor move at different speeds.

Digital

A digital value is usually either on or off. It is a discrete value, and can not be inbetween. This is far less error prone than analog.

IP

Internet protocol. The protocol that allows computors on the internet to communicate.

IP Address

A unique address that every computor on the internet has, the address usually has 4 sets of numbers and look like this: 192.78.1.8

DNS

Domain Name System/Server. It associates the name of a website with the correct IP address.

Bandwidth

A measure of how fast information can traval through a connection in a network. If you have a high bandwidth connection to the internet, you have a fast connection to to internet.
